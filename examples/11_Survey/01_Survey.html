getSurveyFlowJson

Retrieves the survey flow JSON for a specified survey. The result is delivered asynchronously via the surveyflowjsonloaded event listener.

SYNTAX
PresentationPlayer.getSurveyFlowJson(options)


HOW IT WORKS

// 1. Register listener for the response PresentationPlayer.registerEventListener( 'surveyflowjsonloaded', function(data) { var surveyData = JSON.parse(data); // Build survey UI from surveyData } ); // 2. Request the survey flow PresentationPlayer.getSurveyFlowJson({ developerName: 'My_Survey' });
Async pattern: getSurveyFlowJson doesn't return data directly. Register the surveyflowjsonloaded event listener first, then call getSurveyFlowJson.

setSurveyFlowJson

Saves or submits survey flow JSON data. Use 'save' for drafts (can be resumed) or 'submit' to finalize responses and create records.

SYNTAX
PresentationPlayer.setSurveyFlowJson(surveyJson, state)

FULL WORKFLOW EXAMPLE

// Step 1: Load survey PresentationPlayer.registerEventListener( 'surveyflowjsonloaded', function(data) { surveyData = JSON.parse(data); renderSurveyUI(surveyData); } ); PresentationPlayer.getSurveyFlowJson({ developerName: 'Patient_Survey' }); // Step 2: User fills out survey... // Step 3: Save draft (user can resume later) PresentationPlayer.setSurveyFlowJson( surveyData, 'save' ); // Step 4: Submit final answers PresentationPlayer.setSurveyFlowJson( surveyData, 'submit' );
States: 'save' = draft (resumable). 'submit' = final (creates records). The surveyflowjsonpassedtovisit event fires when survey JSON is passed back to the visit.
