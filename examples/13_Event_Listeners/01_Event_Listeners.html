Event Listeners Demo

registerEventListener

Registers a JavaScript function to handle a specific iOS event from the presentation player. Each event can have one handler at a time.

SYNTAX
PresentationPlayer.registerEventListener( iOS_event, handler )

Common Patterns

PAGE LIFECYCLE
// Start tracking when page appears PresentationPlayer.registerEventListener( 'viewappearing', function() { PresentationPlayer.startTrackingPage('myPage'); } ); // Save state when page disappears PresentationPlayer.registerEventListener( 'viewdisappearing', function() { PresentationPlayer.stopTrackingPage(); PresentationPlayer.saveState( JSON.stringify(currentState) ); } );
DISMISS CONTROL
// Handle Visit button press with // long-running operations PresentationPlayer.registerEventListener( 'returntovisitbuttonpress', function() { PresentationPlayer.disableDismiss(); // Do work (upsert, save state, etc.) PresentationPlayer.upsert( [record], 'callback' ); PresentationPlayer.enableDismiss(); } );
Important: The returntovisitbuttonpress event is sent only to the currently open presentation. viewappearing fires each time the page comes into view, including returns from other pages.
