Email Demo

launchApprovedEmail

Opens the Send Email window with the email template that's linked to the current presentation page. If there's no email template ID on the page, nothing happens.

SYNTAX
PresentationPlayer.launchApprovedEmail()
Prerequisite: The presentation page must have an emailTemplateId associated with it. This ID is available in the JSON data at the top level.

EMAIL TEMPLATE FUNCTIONS (for template HTML)

These functions are used inside email template HTML files (not presentation pages):

FETCH DATA IN EMAIL
EmailTemplate.fetchWithParams( 'SELECT Id, Name FROM Account', {'batchSize': 100}, 'getAccountRecordsCallback' );
VALIDATION FAILED
// Prevent sending if data is missing EmailTemplate.fetchValidationFailed( 'Type is missing' );
Email Template Rules: Max 100 records per fetch. Avoid setTimeout, setInterval, and promises. Use parent-based CSS styles (not global). Place scripts in <script type="text/javascript"> tags.

launchEmails

Opens the Send Email window with specific email templates, or all templates available for the selected territory if left blank.

SYNTAX
PresentationPlayer.launchEmails(templates)


Arguments: A comma-separated list of email template names. If left blank, users see all templates available for the selected territory.
