EMAIL ATTACHMENTS

Include PDFs in the ZIP for email attachments. Use the required__ prefix to make them mandatory.

// Required attachment (always sent) required__product_info.pdf // Optional attachment isadditionalcontent_supplement.pdf // PDFs under 3 MB are auto-attached // to the presentation page record

ZIP File Structure

Each presentation page is a ZIP file. Additional content files live alongside the main HTML and thumbnail.

01_ProductDetail.zip ├── 01_ProductDetail.html ├── 01_ProductDetail.jpg ├── isadditionalcontent_study.pdf ├── isadditionalcontent_demo.mp4 ├── isadditionalcontent_tool.html ├── required__brochure.pdf ├── css/ │ └── styles.css ├── js/ │ └── app.js └── images/ └── chart.png

HTML CODE EXAMPLE

PDF LINK
<!-- Link to PDF in ZIP --> <a href="isadditionalcontent_study.pdf"> View Clinical Study </a> <!-- Open in new window --> <a href="isadditionalcontent_study.pdf" target="_blank"> Open PDF </a>

ZIP FILE REQUIREMENTS

Naming: XX_PageName.zip (XX = order) HTML file: Must match ZIP name Thumbnail: Must match ZIP name (.jpg) Max size: Recommended under 20 MB Encoding: UTF-8 for all text files Formats: PDF, MP4, MOV, M4V, HTML
Key rules: The HTML filename must match the ZIP filename. The JPEG thumbnail must also match. The isadditionalcontent_ prefix tells the player to make the file available alongside the main page.
