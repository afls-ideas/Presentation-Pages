Navigation API Summary

goNextPage()

PresentationPlayer.goNextPage()
Move to the next page in order. No-op on last page. Desktop + Mobile

goPreviousPage()

PresentationPlayer.goPreviousPage()
Move to the previous page in order. No-op on first page. Desktop + Mobile

gotoSlide(identifier, slideName, animation)

PresentationPlayer.gotoSlide(id, file, anim)
Jump to any page by ID, name, or external ID. Supports animation: noanimation, swipeleft, swiperight. Desktop + Mobile
This 3-page demo shows: Sequential navigation (Next/Previous), direct jumps (goToSlide from Page 1 to Page 3), and animation options. In production, each page is a separate ZIP file.
